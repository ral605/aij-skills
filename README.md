# aij-skills

Claude Code skills published by [Rod Leach](https://solvx.com) for the AI and Automation Journey community. Practical, battle-tested skills for real business workflows.

---

## Install

Add this marketplace to Claude Code:

```
/plugin marketplace add ral605/aij-skills
```

Then install any skill:

```
/plugin install marketing-clutter-index@aij-skills
```

---

## Available Skills

### marketing-clutter-index

Score and coach any social media post, ad, or marketing graphic on visual clutter and caption clarity. Gives you a scored breakdown across image and caption, a diagnosis, the one highest-leverage fix, and a built-in rewrite — all in one pass.

Triggers automatically when you share a post for feedback, ask "is this too busy?", say "rate my flyer", or ask Claude to clean up a caption.
